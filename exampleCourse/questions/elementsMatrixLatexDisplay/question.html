The element pl-matrix-latex can be used to display 2D numpy arrays and scalars.

Display a 2D array of floats with {{params.sf}} significant figures, by setting presentation-type="sigfig" and digits="{{params.sf}}"

Display a 1D array a = np.array({{params.M}}). To use this element, we first need to wrap the 1D array as a 2D array. For example, we use b = a.reshape((1, M)) to display as a row-vector or c = a.reshape((M, 1)) to display as a column vector:
